# 使用指南

本指南面向第一次使用 `zephyr-mail` 的用户，按常见场景说明如何收信、查信、标记和发信。

## 1. 安装与准备

先在项目根目录放好 `.env`，再构建程序：

```bash
go build -o zephyr-mail ./cmd/zephyr-mail
```

确认你的邮箱服务支持 IMAP 和 SMTP，然后配置好账号、密码、服务器地址和端口。

## 2. 查看收件箱

先检查有没有新邮件：

```bash
./zephyr-mail check --limit 10
```

常用参数：

- `--limit`：最多返回多少封邮件，默认 `10`
- `--recent`：最近时间窗口，例如 `24h`、`2h`、`7d`
- `--unseen`：只查未读邮件，但当前实现要求显式传 `--unseen=true`

## 3. 精确查找邮件

如果你要按发件人、主题或时间过滤，使用 `search`：

```bash
./zephyr-mail search --from alice@example.com --subject 报表 --recent 24h
```

常见组合：

- 未读邮件：`search` 用 `--unseen`，`check` 用 `--unseen=true`
- 已读邮件：`--seen`
- 标记邮件：`--flagged`
- 已回复邮件：`--answered`
- 按 UID 查询：`--uid 10` 或 `--uid 10:20`

## 4. 查看单封邮件

```bash
./zephyr-mail fetch 12345
```

这会按 UID 取回邮件正文、主题、发件人和收件人等信息。

## 5. 标记已读或未读

```bash
./zephyr-mail mark-read 12345
./zephyr-mail mark-unread 12345
```

也可以一次传多个 UID：

```bash
./zephyr-mail mark-read 1 2 3
```

## 6. 查看邮箱列表

```bash
./zephyr-mail list-mailboxes
```

适合先确认服务器上有哪些邮箱目录，再决定 `--mailbox` 参数。

## 7. 查看附件信息

```bash
./zephyr-mail download 12345 --dir .
```

如果指定文件名：

```bash
./zephyr-mail download 12345 --file invoice.pdf
```

当前实现会检查邮件中的附件并返回匹配结果；当指定文件不存在时，会返回可用文件名。

## 8. 发送邮件

```bash
./zephyr-mail send --to bob@example.com --subject "你好" --body "这是一封测试邮件"
```

支持抄送、密送、HTML 和附件：

```bash
./zephyr-mail send \
  --to bob@example.com \
  --cc team@example.com \
  --subject-file subject.txt \
  --html-file mail.html \
  --attach a.pdf,b.png
```

规则：

- `--to` 必填
- `--subject` 和 `--subject-file` 至少提供一个
- `--subject-file` 会覆盖 `--subject`
- `--body-file` 优先于 `--html-file` 和 `--body`

## 9. 测试 SMTP 连通性

```bash
./zephyr-mail test
```

这个命令会使用当前 SMTP 配置发送一封测试邮件，用于确认账号和服务器是否可用。
