# 配置说明

`zephyr-mail` 通过环境变量读取配置，通常放在项目根目录的 `.env` 中。

## 1. 加载规则

程序会按顺序读取：

1. 当前工作目录下的 `.env`
2. 可执行文件所在目录下的 `.env`

如果文件不存在，不会报错；如果 `.env` 格式错误，会直接返回错误。

## 2. IMAP 配置

| 变量 | 默认值 | 说明 |
| --- | --- | --- |
| `IMAP_HOST` | `127.0.0.1` | IMAP 服务器地址 |
| `IMAP_PORT` | `993` | IMAP 端口 |
| `IMAP_USER` | 空 | 登录用户名 |
| `IMAP_PASS` | 空 | 登录密码 |
| `IMAP_TLS` | `false` | 是否使用 TLS |
| `IMAP_REJECT_UNAUTHORIZED` | `true` | 是否校验证书；设为 `false` 可跳过校验 |
| `IMAP_MAILBOX` | `INBOX` | 默认邮箱目录 |

### 校验规则

- `IMAP_USER` 和 `IMAP_PASS` 必填
- `IMAP_HOST` 不填时使用默认值 `127.0.0.1`

## 3. SMTP 配置

| 变量 | 默认值 | 说明 |
| --- | --- | --- |
| `SMTP_HOST` | 空 | SMTP 服务器地址 |
| `SMTP_PORT` | `587` | SMTP 端口 |
| `SMTP_USER` | 空 | 登录用户名 |
| `SMTP_PASS` | 空 | 登录密码 |
| `SMTP_SECURE` | `false` | 是否使用 TLS/安全连接 |
| `SMTP_REJECT_UNAUTHORIZED` | `true` | 是否校验证书 |
| `SMTP_FROM` | 空 | 默认发件人地址 |

### 校验规则

- `SMTP_HOST`、`SMTP_USER`、`SMTP_PASS` 必填

## 4. 布尔值约定

- `IMAP_TLS`、`SMTP_SECURE` 只有在值为 `true` 时才算开启
- `IMAP_REJECT_UNAUTHORIZED`、`SMTP_REJECT_UNAUTHORIZED` 只有在值为 `false` 时才会关闭校验

## 5. 示例 `.env`

```env
IMAP_HOST=imap.example.com
IMAP_PORT=993
IMAP_USER=user@example.com
IMAP_PASS=secret
IMAP_TLS=true
IMAP_REJECT_UNAUTHORIZED=true
IMAP_MAILBOX=INBOX

SMTP_HOST=smtp.example.com
SMTP_PORT=587
SMTP_USER=user@example.com
SMTP_PASS=secret
SMTP_SECURE=true
SMTP_REJECT_UNAUTHORIZED=true
SMTP_FROM=user@example.com
```
